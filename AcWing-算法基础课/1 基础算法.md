## 排序

### 快速排序

**思想：分治**

1. 确定分界点

2. 调整区间

3. 递归处理

**模板：**

我采用了三路快排的模板，因为y总的模板有比较复杂的边界情况，不太好记忆。

关于y总模板的边界情况的详细分析：https://www.acwing.com/solution/content/16777/

```c++
const int N = 1e5 + 10;

int n, A[N];

void quickSort(int l, int r)
{
    if (l >= r) return;

    int key = A[rand() % (r - l + 1) + l];
    int i = l, j = r, k = l;
    while (k <= j)
    {
        if      (A[k] < key) std::swap(A[i++], A[k++]);
        else if (A[k] > key) std::swap(A[j--], A[k]);
        else k++;
    }
    quickSort(l, i - 1);
    quickSort(j + 1, r);
}
```

### 归并排序

**思想：分治**

1. 确定分界点：`mid = (l+r)/2`

2. 递归排序

3. 归并：合二为一

**模板：**

```c++
const int N = 1e5 + 10;

int A[N], B[N];

void mergeSort(int l, int r)
{
    if (l >= r) return;

    int mid = l + r >> 1;
    mergeSort(l, mid), mergeSort(mid + 1, r);

    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r)
    {
        if (A[i] <= A[j]) B[k++] = A[i++];
        else B[k++] = A[j++];
    }
    while (i <= mid) B[k++] = A[i++];
    while (j <= r) B[k++] = A[j++];

    for (k = l; k <= r; k++) A[k] = B[k];
}
```

1. 循环中的`if (a[i] <= a[j])`语句带有等号，排序才是稳定的

## 二分

### 整数二分

必须用于有序数组

**模板：**

```c++
// 区间[l, r]被划分成[l, mid]和[mid + 1, r]时使用：
int bsearch1(int l, int r) 
{
    while (l < r) 
    {
        int mid = l + r >> 1;
        if (check(mid)) r = mid;
        else l = mid + 1;
    }
    return l;
}
// 区间[l, r]被划分成[l, mid - 1]和[mid, r]时使用：
int bsearch2(int l, int r) 
{
    while (l < r) 
    {
        int mid = l + r + 1 >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}
```

1. 先根据二分的目标构思`check(mid)`函数，更新l或r的表达式必须保证不能遗失答案。
2. 再考虑计算mid时，什么时候应该加1什么时候不该加，可以思考一个例子，即当`r = l + 1`，这种情况下每次循环`l, r`的更新必须有变化，如果没有变化则会陷入无限循环。

#### 数的范围

题目：[数的范围](https://www.acwing.com/problem/content/description/791/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;

const int N = 1e5 + 10;

int n, q;
int A[N];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> q;
    for (int i = 0; i < n; i++) cin >> A[i];

    while (q--)
    {
        int x; cin >> x;
        int l = 0, r = n - 1;
        while (l < r)
        {
            int mid = l + r >> 1;
            if (A[mid] >= x) r = mid;
            else l = mid + 1;
        }
        
        if (A[l] != x)
        {
            cout << "-1 -1" << '\n';
            continue;
        }

        int L = l;
        r = n - 1;
        while (l < r)
        {
            int mid = l + r + 1 >> 1;
            if (A[mid] <= x) l = mid;
            else r = mid - 1;
        }

        cout << L << ' ' << l << '\n';
    }

    return 0;
}
```

#### 数的三次方根

题目：[790.数的三次方根](https://www.acwing.com/problem/content/792/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;

const double INF = 1e4;
const double eps = 1e-8;

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    double x; cin >> x;

    double l = -INF, r = INF;
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (mid * mid * mid > x) r = mid;
        else l = mid;
    }

    printf("%.6lf\n", l);
    return 0;
}
```

### 高精度减法

**模板：**

A必须大于等于B

```c++
// A >= B 返回 true
bool cmp(vector<int> &A, vector<int> &B)
{
    if (A.size() != B.size()) return A.size() > B.size();

    for (int i = A.size() - 1; i >= 0; i--)
    {
        if (A[i] > B[i]) return true;
        if (A[i] < B[i]) return false;
    }

    return true;
}

vector<int> sub(vector<int>& A, vector<int>& B) 
{
    vector<int> C;
    int t = 0; // 借位
    for (int i = 0; i < A.size(); i++) 
    {
        t = A[i] - t;
        if (i < B.size()) t -= B[i];
        C.push_back((t + 10) % 10);
        t = t < 0 ? 1 : 0; // 生成借位
    }

    // 消除前导0
    while (C.size() > 1 && C.back() == 0) C.pop_back();

    return C;
}
```

## 前缀和与差分

前缀和和差分是树状数组和线段树的基础。

### 前缀和

#### 一维前缀和

**前缀和公式：**
$$
S_i = \sum_{i=1}^n a_i
$$

数组S和a的下标建议从1开始，并且`S[0] = 0`，则计算$[1, r]$的区间和（即前缀和）的时候可以减少判断



题目：[前缀和](https://www.acwing.com/problem/content/797/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;

const int N = 1e5 + 10;

int n, m;
int ps[N];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> ps[i];
    for (int i = 1; i <= n; i++) ps[i] += ps[i - 1];

    while (m--)
    {
        int l, r; cin >> l >> r;
        cout << ps[r] - ps[l - 1] << '\n';
    }

    return 0;
}
```



#### 二维前缀和

计算矩阵中的一个矩形区域的部分和。以下公式应该画图理解。

**二维前缀和计算公式：**

`S[i][j] = S[i - 1][j] + S[i][j - 1] - S[i - 1][j - 1] + a[i][j]`

**部分和计算公式：**

以$(x_1,y_1)$为左上角，$(x_2, y_2)$为子矩阵的部分和

`S[x2][y2] - S[x1 - 1][y2] - S[x2][y1 - 1] + S[x1 - 1][y1 - 1]`



题目：[子矩阵的和](https://www.acwing.com/problem/content/798/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;

const int N = 1010;

int n, m, q;
int ps[N][N];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> m >> q;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
        {
            cin >> ps[i][j];
            ps[i][j] += -ps[i - 1][j - 1] + ps[i - 1][j] + ps[i][j - 1];
        }

    while (q--)
    {
        int x1, y1, x2, y2; cin >> x1 >> y1 >> x2 >> y2;
        cout << ps[x2][y2] - ps[x2][y1 - 1] - ps[x1 - 1][y2] +
                ps[x1 - 1][y1 - 1] << '\n';
    }

    return 0;
}
```

### 差分

#### 一维差分

差分是前缀和的逆运算，如果数组B是数组A的差分，那么数组A是数组B的前缀和。

**差分计算公式：**

`B[i] = A[i] - A[i-1]`

**插入操作：**

对于数组A的$[l, r]$区间的每一个数字加上c在数组B上等同于`B[l] += c, B[r+1] -= c`

在读入数组A时，可以将读一个数字等同于插入一个数字，即读入`A[i]`等同于在区间$[i,i]$上插入`a[i]`



题目：[差分](https://www.acwing.com/problem/content/799/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;

const int N = 1e5 + 10;

int n, m;
int A[N], B[N];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> A[i];
    for (int i = 1; i <= n; i++) B[i] = A[i] - A[i - 1];

    while (m--)
    {
        int l, r, c; cin >> l >> r >> c;
        B[l] += c, B[r + 1] -= c;
    }

    for (int i = 1; i <= n; i++) A[i] = A[i - 1] + B[i];
    for (int i = 1; i <= n; i++) cout << A[i] << ' ';
    cout << '\n';

    return 0;
}
```

#### 二维差分

**插入操作：**

给以$(x_1, y_1)$为左上角，$(x_2, y_2)$为右下角的子矩阵中的所有元素加上c。可以画图理解

```c++
B[x1][y1] += c; 
B[x2 + 1][y1] -= c;
B[x1][y2 + 1] -= c;
B[x2 + 1][y2 + 1] += c;
```



题目：[差分矩阵](https://www.acwing.com/problem/content/800/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;

const int N = 1010;

int n, m, q;
int B[N][N];

void insert(int x1, int y1, int x2, int y2, int c)
{
    B[x1][y1] += c;
    B[x1][y2 + 1] -= c;
    B[x2 + 1][y1] -= c;
    B[x2 + 1][y2 + 1] += c;
}

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> m >> q;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
        {
            int c; cin >> c;
            insert(i, j, i, j, c);
        }

    while (q--)
    {
        int x1, y1, x2, y2, c;
        cin >> x1 >> y1 >> x2 >> y2 >> c;
        insert(x1, y1, x2, y2, c);
    }

    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            B[i][j] += -B[i - 1][j - 1] + B[i - 1][j] + B[i][j - 1];
    
    for (int i = 1; i <= n; i++)
    {
        for (int j = 1; j <= m; j++)
            cout << B[i][j] << ' ';
        cout << '\n';
    }

    return 0;
}
```

### 最长连续不重复子序列

题目：[799. 最长连续不重复子序列](https://www.acwing.com/problem/content/801/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using LL = long long;

const int N = 1e5 + 10;

int n;
int A[N], ht[N];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n;
    for (int i = 1; i <= n; i++) cin >> A[i];

    int res = 0;
    for (int i = 1, j = 1; i <= n; i++)
    {
        ht[A[i]]++;
        while (ht[A[i]] > 1)
        {
            ht[A[j]]--;
            j++;
        }
        res = std::max(res, i - j + 1);
    }

    cout << res << '\n';
    return 0;
}
```

## 离散化

用作稀疏数组的下标映射

**模板：**

```c++
vector<int> alls; // 存储所有待离散化的值
sort(alls.begin(), alls.end()); // 将所有值排序
alls.erase(unique(alls.begin(), alls.end()), alls.end());   // 去掉重复元素

// 二分求出x对应的离散化的值
int find(int x) // 找到第一个大于等于x的位置
{
    int l = 0, r = alls.size() - 1;
    while (l < r)
    {
        int mid = l + r >> 1;
        if (alls[mid] >= x) r = mid;
        else l = mid + 1;
    }
    return r + 1; // 映射到1, 2, ...n
}
```

**unique函数的实现：**

```c++
typedef vector<int>::iterator iter;
iter myunique(iter begin, iter end) 
{
    iter j = begin;
    for (iter i = begin; i != end; i++) 
    {
        if (i == begin || *i != *(i - 1)) 
        {
            *j = *i;
            j++;
        }
    }
    return j;
}
```



### 区间和

题目：[区间和](https://www.acwing.com/problem/content/804/)

题解：https://www.acwing.com/solution/content/13511/

注意，离散后的坐标最多有$3N$个；

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>
#include <vector>
#include <unordered_map>

using std::cin, std::cout;
using LL = long long;
using PII = std::pair<int, int>;

const int N = 1e5 + 10;

int n, m;
PII query[N];
int ps[N * 3];

std::vector<int> alls;

int find(int x)
{
    return std::lower_bound(alls.begin(), alls.end(), x) - alls.begin() + 1;
}

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> m;
    std::unordered_map<int, int> ht;
    for (int i = 0; i < n; i++)
    {
        int x, c; cin >> x >> c;
        ht[x] += c;
        alls.push_back(x);
    }
    for (int i = 0; i < m; i++)
    {
        int l, r; cin >> l >> r;
        query[i] = {l, r};
        alls.push_back(l);
        alls.push_back(r);
    }

    sort(alls.begin(), alls.end());
    alls.erase(unique(alls.begin(), alls.end()), alls.end());

    for (auto [x, c] : ht) ps[find(x)] = c;
    for (int i = 1; i <= alls.size(); i++) ps[i] += ps[i - 1];

    for (int i = 0; i < m; i++)
    {
        auto [l, r] = query[i];
        cout << ps[find(r)] - ps[find(l) - 1] << '\n';
    }

    return 0;
}
```

还可以用`std::map`写离散化，效率低一些，但更简洁。

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>
#include <map>
#include <unordered_map>

using std::cin, std::cout;
using LL = long long;
using PII = std::pair<int, int>;

const int N = 1e5 + 5;

int n, m;
PII query[N];
int tot;
std::map<int, int> disc;
int ps[N * 3];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n >> m;
    std::unordered_map<int, int> ht;
    for (int i = 0; i < n; i++)
    {
        int x, c; cin >> x >> c;
        ht[x] += c;
        disc[x] = 1;
    }
    for (int i = 0; i < m; i++)
    {
        int l, r; cin >> l >> r;
        query[i] = {l, r};
        disc[l] = disc[r] = 1;
    }

    for (auto &item : disc) item.second = ++tot;

    for (auto [x, c] : ht) ps[disc[x]] = c;
    for (int i = 1; i <= tot; i++) ps[i] += ps[i - 1];

    for (int i = 0; i < m; i++)
    {
        auto [l, r] = query[i];
        cout << ps[disc[r]] - ps[disc[l] - 1] << '\n';
    }

    return 0;
}
```



## 区间合并

题目：[区间合并](https://www.acwing.com/problem/content/805/)

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <algorithm>

using std::cin, std::cout;
using LL = long long;
using PII = std::pair<int, int>;

const int N = 1e5 + 5;

int n;
PII range[N];

int main()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    cin >> n;
    for (int i = 0; i < n; i++) 
        cin >> range[i].first >> range[i].second;

    sort(range, range + n);

    int cnt = 0, last = range[0].first - 1;
    for (int i = 0; i < n; i++)
    {
        if (last < range[i].first) cnt++;
        last = std::max(last, range[i].second);
    }

    cout << cnt << '\n';
    return 0;
}
```
